## 🎯 Project Vision

**LevelUp** is a gamified self-improvement app that transforms personal goals into an engaging adventure. Users can track habits, complete quests, earn XP, and level up while achieving their personal development objectives.

### Core Value Proposition

- **Gamified Experience**: Turn self-improvement into an engaging game
- **Personalized Journey**: Adapt to individual user preferences and goals
- **Progress Tracking**: Visual progress and achievement system
- **Community Support**: Social features and accountability

## 🛠️ Quick Start

### Development Environment

```bash
# Clone the repository
git clone <repository-url>
cd Self-help

# Backend setup
cd my-new-app/backend
npm install
npm run dev

# Frontend setup
cd ../
npm install
npm start
```

### Testing

```bash
# Backend tests
cd my-new-app/backend
npm run test

# Frontend tests
cd ../
npm run test

# Full application test
npm run test:full
```
